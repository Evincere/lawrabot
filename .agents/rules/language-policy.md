# Política de Idiomas

1. **Conversaciones**: En el chat con semper (desarrollador - usuario) siempre se utilizará el idioma español para las conversaciones.
2. **Código**: En el código siempre se utilizará el idioma inglés.
3. **Comentarios**: Los comentarios en el código se escribirán en idioma español.
